# COAL Practice Tasks (Assembly Language)

This repository contains my practice work for the COAL (Computer Organization and Assembly Language) course.

## 📌 About
These files include various assembly language programs written for learning and practice purposes. The tasks help improve understanding of low-level programming, CPU operations, memory management, and basic algorithm implementation in assembly language.

## 🧠 Topics Covered
- Basic Assembly Instructions
- Loops and Conditions
- Arrays and Strings
- Memory Access
- Input/Output Operations
- Simple Problem Solving in Assembly

## 🛠️ Tools Used
- MASM / TASM (or your assembler name)
- DOSBox (if used)
- Windows Environment

## 📂 File Types
- `.asm` source files
- `.inc` include files (if any)

## 🎯 Purpose
To strengthen concepts of computer organization and improve assembly programming skills through hands-on practice.
